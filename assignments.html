Assignment 1

Data Collection and Analysis: For this assignment, I contributed to gathering data on elevator arrival times at the ground floor in the CDS building. Using this dataset, I determined the optimal waiting position to reduce the expected walking distance to the next arriving elevator.

Assignment 2

Developed an interactive web application demonstrating the KMeans clustering algorithm with various initialization methods. The project includes dynamic visualizations, a responsive interface, and is hosted locally for demonstration.

Assignment 3

In this assignment, I used Singular Value Decomposition (SVD) on the MNIST dataset to examine the impact of dimensionality reduction on logistic regression accuracy and training time. I found that around 100 components capture the essential information while balancing accuracy and computational cost, with visualizations showing key data patterns.

Assignment 4

I developed a Latent Semantic Analysis (LSA) search engine that processes user queries and retrieves the most relevant documents based on cosine similarity. This web application uses dimensionality reduction with SVD to enhance the accuracy of search results. It includes a bar chart visualization to show similarity scores for the top retrieved documents.

Assignment 5

I developed a customer churn prediction model for a bank using KNN. The project involved data preprocessing, feature engineering, and dimensionality reduction to improve model performance. This predictive model helps the bank identify high-risk customers and take action to reduce churn.

Assignment 6

In this assignment, I explore how changing parameters impacts linear regression by building an interactive webpage. I adjust the sample size, noise, and other settings to see how randomness affects the slope and intercept, even when there's no actual relationship between the variables.

Assignment 7

Built on my previous work by implementing hypothesis testing and confidence intervals in linear regression through simulations. My interactive webpage now lets users test the slope or intercept of a regression line and generate confidence intervals based on simulation results.

Assignment 8

In this assignment, I explored how shifting clusters affects the performance of a logistic regression model. I generated datasets, fitted models, and analyzed key parameters like coefficients, decision boundaries, and logistic loss.

Assignment 9

I built an interactive web application to visualize neural network behavior with Tanh, ReLU, and Sigmoid activations. The app generates dynamic 3D plots, decision boundaries, and gradient visualizations, showcasing the training process in real time. This project highlights my skills in Python, machine learning, and data visualization.

Assignment 10

I created a simplified version of Google Image Search with a graphical interface that supports both text and image queries. Users can retrieve the top 5 most relevant images with similarity scores, adjusting query weights for combined searches. Additionally, the system allows switching between embeddings from principal components or CLIP embeddings for image-based searches.
